Selection Sort

Selection sort is a sorting algorithm that works by repeatedly finding the minimum element (or maximum element) in an unsorted array and swapping it with the first element of the unsorted array. This process is repeated until the entire array is sorted.

Here is the algorithm for selection sort:

  1. Initialize the variable `minIndex` to 0. This represents the index of the smallest element in the array.
  2. Iterate through the array, starting at index 1. For each iteration, compare the element at the current index to the element at `minIndex`. If the element at the current index is smaller than the element at `minIndex`, then update `minIndex` to the current index.
  3. Once we have iterated through the entire array, the variable `minIndex` will contain the index of the smallest element in the array. We can then swap the element at `minIndex` with the element at the first index of the array. This will sort the first element of the array in ascending order.
  4. Repeat steps 2 and 3 for the remaining elements in the array. Each time we iterate through the array, we will swap the smallest element with the element at the first index. This will continue until the entire array is sorted.
