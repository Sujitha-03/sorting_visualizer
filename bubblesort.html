Bubble Sort

Bubble sort is a simple sorting algorithm that works by repeatedly comparing adjacent elements in an array and swapping them if they are in the wrong order. The algorithm starts at the beginning of the array and compares the first two elements. If the first element is greater than the second element, then the two elements are swapped. The algorithm then continues to the next two elements and repeats the process. This continues until the algorithm reaches the end of the array.
