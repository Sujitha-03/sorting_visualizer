Merge Sort

Merge sort is a sorting algorithm that works by dividing the unsorted array into two halves, sorting each half recursively, and then merging the two sorted halves together. This process is repeated until the entire array is sorted.

Here is the algorithm for selection sort:

  1. Divide: Divide the unsorted array into two halves.To divide the unsorted array into two halves, we can use the following code:
    let mid = arr.length / 2;
    let left = arr.slice(0, mid);
    let right = arr.slice(mid);
    This code will create two new arrays, left and right, that contain the left and right halves of the original array, respectively.
  2. Sort: Recursively sort each half of the array.
  3. Merge: Merge the two sorted halves of the array together.
